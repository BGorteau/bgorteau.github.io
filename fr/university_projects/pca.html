3 Program a cr function that centers and reduces an array of data.

The first step is to calculate the standard deviation of each variable in the input dataset.

The standard deviation of a variable \(X\) is compute by the following formula :

\[\sigma(X) = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_i - \bar{X})^2}\]

ecartype <- function(x){
  sqrt(mean((x-mean(x))^2))
}

When standard deviation is computed, we can center and reduce our data with the followig formula :

\[X' = \frac{X - \bar{X}}{\sigma(X)}\]

cr <- function(matrice){
  n <- nrow(matrice)
  moy <- apply(matrice, FUN = mean, MARGIN = 2)
  moy <- matrix(rep(moy, n), nrow = n, byrow = T)
  
  e_type <- apply(matrice, FUN = ecartype, MARGIN = 2)
  e_type <- matrix(rep(e_type,n), nrow = n, byrow = T)
  return((matrice-moy)/e_type)
}

4 Program an acp function to perform PCA calculations

Our function need to return :

  • Axis-by-axis inertia calculation
  • Axis coordinates of variables (for 3 axes)
  • Coordinates of individuals on axes (for 3 axes)

The first step in our function is to calculate the covariance and correlation matrix of our data :

  • Covariance : \[\text{Cov}(X, Y) = \frac{1}{n} \sum_{i=1}^{n} (X_i - \bar{X})(Y_i - \bar{Y})\] \[\Sigma = \begin{bmatrix} \text{Cov}(X_1, X_1) & \text{Cov}(X_1, X_2) & \cdots & \text{Cov}(X_1, X_n) \\ \text{Cov}(X_2, X_1) & \text{Cov}(X_2, X_2) & \cdots & \text{Cov}(X_2, X_n) \\ \vdots & \vdots & \ddots & \vdots \\ \text{Cov}(X_n, X_1) & \text{Cov}(X_n, X_2) & \cdots & \text{Cov}(X_n, X_n) \end{bmatrix} \]
  • Correlation : \[\rho(X, Y) = \frac{\text{Cov}(X, Y)}{\sigma_X \sigma_Y}\] \[R = \begin{bmatrix} 1 & \rho(X_1, X_2) & \cdots & \rho(X_1, X_n) \\ \rho(X_2, X_1) & 1 & \cdots & \rho(X_2, X_n) \\ \vdots & \vdots & \ddots & \vdots \\ \rho(X_n, X_1) & \rho(X_n, X_2) & \cdots & 1 \end{bmatrix}\]

Once these two matrix have been calculated, the inertia of each axis is calculated from the eigenvalues of the covariance matrix.

  • eigenvalues : The eigenvalues are determined by solving the following system : \[\det(X - \lambda I) = 0 \] with \(X\) the data, \(\lambda\) the eigenvalues and \(I\) the identity matrix.
  • inertia : \[\text{In} = \frac{\lambda}{\sum_{j=1}^{p} \lambda_j}\times 100\] with \(\lambda = (\lambda_{1}...\lambda_{n})\)

We then calculate the coordinates of the variables on the axes, always using the eigenvalues but also the eigenvectors.

  • eigenvectors : The eigenvectors are determined by solving the following system : \[X \cdot v_j = \lambda_j \cdot v_j\] with \(X\) the data, \(v_j\) the eigenvector you are looking for and \(\lambda_j\) the eigenvalue corresponding to the eigenvector \(v_j\).
  • coordinates of the variables on the axes : \[\text{Coor}_j = \sqrt{\lambda_j}*v_j\]

We can now calculate the coordinates of the individuals on the axes. The coordinates of the axis \(i\) is calculate by the following formula : \[\text{Coord-ind}_i = \sum_{j = 1}^{p}X'_j \times v_{j,i}\]

acp <- function(tablecr){
  mat_cor <- cor(tablecr)
  mat_cov <- cov(tablecr)
  # Calculating inertia
  inertie <- (eigen(mat_cov)$values/sum(eigen(mat_cov)$values))*100
  inertie <- t(as.matrix(inertie))
  colnames(inertie) <- c(1:length(inertie))
  row.names(inertie) <- "Axis inertia (in %)"
  # Variable coordinates
  mat_eig <- eigen(mat_cor)
  mat_val_propres <- NULL
  for(j in 1:ncol(tablecr)){
    rf <- sqrt(mat_eig$values[j])*mat_eig$vectors[,j]
    mat_val_propres <- cbind(mat_val_propres,rf)
  }
  row.names(mat_val_propres) <- colnames(tablecr)
  coord_var <- mat_val_propres[,1:3]
  colnames(coord_var) <- c("axis1","axis2","axis3")
  # Coordinates of individuals on axes
  axe1 <- 0
  axe2 <- 0
  axe3 <- 0
  for (i in 1:ncol(tablecr)){
    axe1 <- axe1+tablecr[,i]*eigen(mat_cor)$vectors[i,1]
    axe2 <- axe2+tablecr[,i]*eigen(mat_cor)$vectors[i,2]
    axe3 <- axe3+tablecr[,i]*eigen(mat_cor)$vectors[i,3]
  }
  coord_indiv <- data.frame(axe1,axe2,axe3)
  row.names(coord_indiv) <- rownames(tablecr)
  return(list("inertia" =inertie ,"var"=coord_var,"indiv"=coord_indiv))
}

5 Program a plotacp function to produce PCA graphs

The plotacp function allows us to visualize individuals and variables according to their coordinates on two selected axes.

plotacp <- function(X,axe1,axe2,type){
  if(type=="ind"){
    plot(X$indiv[,axe1],X$indiv[,axe2],pch=20,main="PCA graph for individuals",xlab="",ylab="",xlim=c(min(X$ind[,axe1])-0.5,max(X$ind[,axe1])+0.5),ylim=c(min(X$ind[,axe2])-0.5,max(X$ind[,axe2])+0.5))
    title(xlab = paste("Dim",axe1,"(",round(X$inertia[1,axe1],2),"%)"),ylab =paste("Dim",axe2,"(",round(X$inertia[1,axe2],2),"%)"),adj=1,line=2.2)
    grid(NULL,NULL,lty=2,lwd=1)
    abline(h=0)
    abline(v=0)
    text(X$indiv[,axe1],X$indiv[,axe2],labels=row.names(X$indiv),cex=0.9,pos=1)
  }
  if(type=="var"){
    plot(X$var[,axe1],X$var[,axe2],xlim=c(-1,1),ylim=c(-1,1),type="n",main="PCA graph for variables",xlab="",ylab="")
    title(xlab = paste("Dim",axe1,"(",round(X$inertia[1,axe1],2),"%)"),ylab =paste("Dim",axe2,"(",round(X$inertia[1,axe2],2),"%)"),adj=1,line=2.2)
    grid(NULL,NULL,lty=2,lwd=1)
    segments(-1,0,1,0,lty=2)
    segments(0,-1,0,1,lty=2)
    draw.circle(0,0,1)
    text(X$var[,axe1],X$var[,axe2],labels=row.names(X$var),cex=0.9,pos=1)
    for(i in 1:nrow(X$var)){
      arrows(0,0,X$var[i,axe1],X$var[i,axe2],length = 0.1)
    }
  }
}

6 Program an indsup function to calculate the coordinates of additional individuals (for 3 axes)

In this section, we build on the previous steps. - We determine which individuals in our dataset are additional individuals. - We center and reduce the additional individuals from the data from which we subtract the additional individuals. - We calculate the coordinates of the extra individuals on the axes from the eigenvectors of the data without the extra individuals.

indsup <- function(data_frame,ind_sup){
  table_ind_sup <- data_frame[ind_sup,]
  n <- nrow(table_ind_sup)
  data_frame_2 <- data_frame[-ind_sup,]
  moy_table <- apply(data_frame_2,FUN=mean,MARGIN = 2)
  moy_table <-  matrix(rep(moy_table, n), nrow = n, byrow = T)
  ec_type_table <- apply(data_frame_2,FUN=ecartype, MARGIN = 2)
  ec_type_table <- matrix(rep(ec_type_table,n), nrow = n, byrow = T)
  ind_sup_cr <- (table_ind_sup-moy_table)/ec_type_table
  axe1 <- 0
  axe2 <- 0
  axe3 <- 0
  mat_vect_propre <- eigen(cov(cr(data_frame_2)))$vectors
  for (i in 1:ncol(table_ind_sup)){
    axe1 <- axe1+ind_sup_cr[,i]*mat_vect_propre[i,1]
    axe2 <- axe2+ind_sup_cr[,i]*mat_vect_propre[i,2]
    axe3 <- axe3+ind_sup_cr[,i]*mat_vect_propre[i,3]
  }
  coord_indiv <- data.frame(axe1,axe2,axe3)
  row.names(coord_indiv) <- row.names(data_frame[ind_sup,])
  return(coord_indiv)
}
